Stick Widgets inside their rows for Siteorigin Page Builder

== Description ==

This plugin creates a options panel inside siteOrigin Widgets called 'Sticky', using it's options you can stick widgets inside theirs rows. Made using the great Jquery Plugin **Sticky Kit**

Visit the Jquery plugin site here [Sticky Kit](http://leafo.net/sticky-kit/, 'Sticky Kit')



== Installation ==



1. Upload `sticky-siteorigin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
